# Professional Portfolio Website

A modern, responsive portfolio website showcasing Wessel Stam's skills and experience as an Information Security Officer and Software Developer.

## Features

- **Modern Design**: Clean, professional design with gradient hero section
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Smooth scrolling between sections with active link highlighting
- **Interactive UI**: Hover effects, animations, and transitions
- **Skills Showcase**: Organized display of technologies and skills
- **Project Highlights**: Featured projects with descriptions and tags
- **GitHub Integration**: Live GitHub statistics display
- **Contact Section**: Easy access to LinkedIn and GitHub profiles
- **Mobile Menu**: Hamburger menu for mobile devices
- **Scroll to Top**: Convenient button to return to the top of the page
- **Performance Optimized**: Lazy loading images and debounced scroll events
- **Accessible**: Semantic HTML, ARIA labels, and keyboard navigation support

## Sections

1. **Hero**: Introduction with name, title, and call-to-action buttons
2. **About**: Personal description and core values (Security First, Clean Code, Innovation)
3. **Skills**: Technologies organized by categories (Languages, Frameworks, Databases, Tools, OS, Security)
4. **Projects**: Featured projects with descriptions and technology tags
5. **Contact**: Links to professional profiles

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript**: Interactive features and smooth scrolling
- **Font Awesome**: Icons for social media and sections

## Deployment

### GitHub Pages (Recommended)

1. This repository is already set up as a GitHub profile repository
2. The `index.html` file will be automatically served at: `https://Wessel-Stam.github.io/Wessel-Stam/`
3. Go to repository Settings → Pages → Source: Deploy from branch → Branch: Select your branch

### Local Development

1. Clone the repository
2. Open `index.html` in a web browser, or
3. Use a local server:
   ```bash
   python3 -m http.server 8080
   ```
   Then navigate to `http://localhost:8080`

### Other Hosting Options

- **Netlify**: Drag and drop the files to Netlify
- **Vercel**: Connect the GitHub repository
- **Any Web Host**: Upload `index.html`, `styles.css`, and `script.js` to your web hosting

## Customization

### Update Personal Information

- **Name & Title**: Edit the hero section in `index.html`
- **About Text**: Modify the about section content
- **Skills**: Add or remove skill tags in the skills section
- **Projects**: Update project cards with your own projects
- **Social Links**: Update LinkedIn and GitHub URLs

### Styling

- **Colors**: Modify CSS variables in `styles.css` (`:root` section)
- **Fonts**: Change the font-family in the `body` selector
- **Layout**: Adjust grid and flexbox properties for different layouts

### Adding Sections

1. Add new section HTML in `index.html`
2. Add navigation link in the navbar
3. Style the section in `styles.css`
4. Add smooth scrolling support in `script.js` (automatically works)
